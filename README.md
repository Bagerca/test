# Bendy Developer Feed Aggregator 🎬🖋️

Автономный веб-сервис и парсер для мониторинга обновлений и твитов команды разработчиков серии игр **Bendy** (`#BATIM`, `#BATDR`, `#BendyTHECAGE`). 

Проект агрегирует свежие публикации из Twitter/X без использования официального платного API и отображает их в удобном, адаптивном веб-интерфейсе.

---

## 🚀 Основные особенности

### ⚙️ Backend (Python 3)
* **Мультиуровневый скрейпинг (God-Mode Bypass):** Автоматический обход приватности и жестких лимитов Twitter/X с использованием гибридной системы обхода:
  * **Tier 1:** `Sotwe API` — сбор хронологической ленты через скрытые эндпоинты.
  * **Tier 2:** `Twitter Syndication API` — резервный сбор через официальный сервис виджетов.
* **Атомарная запись данных:** Запись `feed.json` происходит через временный файл (`feed.json.tmp`), что исключает повреждение или чтение незавершенного кэша веб-интерфейсом.
* **Без сторонних зависимостей:** Скрипт работает исключительно на стандартной библиотеке Python (`urllib`, `re`, `json`, `datetime`), не требуя выполнения `pip install`.
* **Умный нормализатор дат:** Преобразует часовые пояса и специфичные форматы дат Twitter в единый ISO-стандарт (ISO 8601) для безупречной хронологической сортировки.

### 🎨 Frontend (JavaScript ES6 / HTML5 / CSS3)
* **Современный UI/UX:** Адаптивная карточная верстка в стиле ленты социальных сетей, адаптированная под ПК, планшеты и мобильные устройства.
* **Мгновенный поиск:** Фильтрация постов по ключевым словам и авторам с поддержкой оптимизации **Debounce** (снижает нагрузку на браузер при быстром вводе).
* **Безопасность (XSS Protection):** Безопасный рендеринг контента через DOM-свойства `.textContent` во избежание инъекций кода.
* **Устойчивость к сбоям:** Автоматические фоллбеки для битых аватарок (SVG-заглушки) и корректная обработка отсутствующих медиафайлов/картинок.

## 🏁 Быстрый запуск

### 1. Обновление ленты (Backend)
Запустите скрипт сборщика из консоли:

```bash
python scraper.py
```
*Скрипт соберет свежие данные, отсортирует их по времени и атомарно обновит файл `feed.json`.*

### 2. Просмотр ленты (Frontend)
Просто откройте файл `index.html` в любом браузере:
* **Вариант А:** Двойной клик по файлу `index.html`.
* **Вариант Б (Рекомендуется):** Запуск через локальный сервер (например, расширение *Live Server* в VS Code или `python -m http.server 8000`).
